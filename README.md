# codefest-gamma-ai

# Usage

Build the library

```python
python setup.py bdist_wheel
```

Install the library:

```python
pip install dist/gamma-0.1.0-py3-none-any.whl
```


## **Task 1: Text**

The first task this library does is the entity detection from newspaper's articles.
